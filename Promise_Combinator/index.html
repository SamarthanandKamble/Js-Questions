Exploring Promise Combinator

  1. Promise.all()

    If any one of the promises failed, it will return the failed promise else it wait till the last promise to fulfill and the return the fulfilled promises array.

  2. Promise.allSettled()

    Irrespective of the state of the promise, it will return an array consisting of promises and their state.

  3. Promise.race()

    As the name suggests, it is like a race.It returns the result of the promise which is executing early ,irrespective of the state.

  4. Promise.any()

    It returns the first fulfilled promise, ignoring if some of the promises are getting rejected. But it will return an aggregated array if all the promises are getting rejected.
